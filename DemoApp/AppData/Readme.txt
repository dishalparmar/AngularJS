To run this project
-------------------

Website
-------
1) Open IIS
2) Create new site "AppData"
3) Physical path - DemoApp\AppData\websiteapp\app (depending on where you save this on your local)
4) Site binding use port 8000  


Acess Data
-------
1) Open IIS
2) Create new site "AppData"
3) Physical path - DemoApp\AppData\AppData (depending on where you save this on your local)
4) Select Application pool ".NET v4.5"
